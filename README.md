# plages-du-maroc

#Ce projet s’inscrit dans le cadre des travaux pratiques du module Web Mapping de la 3ème année cycle ingénieur de la formation en Sciences géomatiques et Ingénierie Topographique. L’objectif est de réaliser un Géoportail sur les plages marocaines, et de synthétiser les différentes données y relatives.
La conception du Géoportail a été orientée de telle sorte à répondre aux exigences du projet, une version destinée au grand public ET/OU aux preneurs de décision va prendre en compte les différentes remarques des utilisateurs afin de réaliser un produit complet sur tous les aspects.
